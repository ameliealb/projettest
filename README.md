#kercode-projet
***General Introduction***
This dataset contains data collected during crack growth experiments at GRETA VANNES, as part of Amelie BOURDIN project (November 2025)
It is being made public both to act as supplementary data for publications and the project of Amelie BOURDIN and in order for other researchers to use this data in their own
work.
The data in this data set was collected in GRETA VANNES in November 2025.
This research project was made possible by GRETA VANNES.
***Purpose of the test campaign***
The purpose of the project is to test Git and GitHub with Linux.
***Test equipment***
All tests were performed on a HP computer.
